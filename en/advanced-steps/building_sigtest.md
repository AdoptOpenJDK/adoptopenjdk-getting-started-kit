Quick build instructions <br/>
```
$ svn checkout https://svn.java.net/svn/sigtest~svn/trunk [dead-link]

$ cd code/build

$ ant build -Djdk5.home=/path/to/jdk1.5.latest \
-Djdk8.home=/path/to/jdk8.latest \
-Dmvn2.exe=/path/to/latest/bin/mvn
```
