
# JMH: Java Microbenchmarking Harness

JMH est un outil de microbenchmarking pour Java et autres langages de la JVM. 

Un microbenchmark est une tentative pour mesurer la performance d'une "micro" portion du code. Ces tests sont typiquement de l'ordre de la nanoseconde pour des milliers d'exécutions. Le code testé effectue habituellement une petite tâche. 

Microbenchmarking n'est pas du profilage ! 

Quand vous profilez, vous travaillez sur l'application dans son intrégralitée soit en production soit dans un environement équivalent.

Quand vous microbenchmarkez, vous obtenez un résultat fictif puisque vous êtes dans un environement très différent de l'environement de production. 

http://openjdk.java.net/projects/code-tools/jmh/
