# Zainstaluj tree (przydatne)

Użyj jedną z instukcji poniżej dla odpowiedniego środowiska.

*Linux (Ubuntu/Debian)*
```
sudo apt-get install tree
```

*Mac OS*

```
brew install tree
sudo port install tree
fink install tree
```
