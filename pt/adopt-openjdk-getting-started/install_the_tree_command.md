# Instale o comando tree (util)

Use uma das instrucoes abaixo para instalar o comando tree de acordo com seu sistema operacional.

*Linux (Ubuntu/Debian)*
```
sudo apt-get install tree
```

*Mac OS*

```
brew install tree
sudo port install tree
fink install tree
```
